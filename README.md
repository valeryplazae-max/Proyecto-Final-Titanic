# Proyecto-Final-Titanic

## Descripción general del proyecto  
Este proyecto busca poner en práctica el análisis de datos utilizando **Python** y la herramienta **Pandas**. A partir del conocido conjunto de datos del *Titanic*, el grupo se propuso descubrir patrones y probabilidades sobre la supervivencia de los pasajeros.  

Durante el proceso, se realizó una exploración, limpieza y estudio de la información, acompañada de gráficos y conclusiones que ayudan a comprender mejor lo ocurrido en el histórico naufragio.  

El trabajo se desarrolló de manera **colaborativa a través de GitHub**, lo que nos permitió aprender sobre organización, control de versiones y trabajo en equipo, tal como se hace en proyectos reales de ciencia de datos.

## Objetivos del trabajo  
1. Aprender a organizar un proyecto colaborativo en GitHub de manera ordenada y clara.  
2. Analizar los datos del Titanic para descubrir tendencias y factores que influyeron en la supervivencia de los pasajeros.  
3. Elaborar visualizaciones y conclusiones que reflejen los hallazgos obtenidos.  
4. Documentar el proceso de trabajo en equipo, mostrando la colaboración y el uso de herramientas digitales.  

## Estructura del proyecto  
El proyecto está organizado en carpetas que facilitan el acceso a cada parte del trabajo:  
├── README.md → Descripción general del proyecto
├── data/ → Información original del Titanic
├── notebooks/ → Cuadernos con análisis y pruebas
├── src/ → Programas en Python usados para procesar los datos
├── outputs/ → Resultados obtenidos (tablas y gráficos)
└── docs/ → Informe final con las conclusiones del proyecto
