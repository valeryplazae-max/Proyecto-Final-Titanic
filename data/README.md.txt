#  Carpeta `data`

Esta carpeta está destinada exclusivamente a almacenar los **archivos originales descargados desde Kaggle**.  
Por política del repositorio, **no se deben subir archivos CSV ni datos pesados** a GitHub.

---

## Enlace de descarga

Los archivos provienen del concurso **"Titanic: Machine Learning from Disaster"** disponible en Kaggle:  
 [https://www.kaggle.com/competitions/titanic/data](https://www.kaggle.com/competitions/titanic/data)

Para descargarlos:
1. Inicia sesión en tu cuenta de Kaggle.  
2. Dirígete al enlace anterior.  
3. Haz clic en **“Download All”** para obtener el archivo `.zip` con los datasets.

---

## Archivos requeridos

Una vez descomprimido el `.zip`, asegúrate de colocar los siguientes archivos dentro de esta carpeta `data/`:

- `train.csv` → Datos de entrenamiento con la variable objetivo `Survived`.  
- `test.csv` → Datos de prueba sin la columna `Survived`.  
- *(Opcional)* `gender_submission.csv` → Ejemplo de formato de predicciones.
